Sistema Bancario en Java:
Aplicación bancaria desarrollada en Java con NetBeans que simula las operaciones básicas de un banco.

Funcionalidades:
Inicio de sesión
Retiros y depósitos
Transferencias entre cuentas
Solicitud y pago de préstamos

Tecnologías usadas:
Java
NetBeans


¿Cómo ejecutarlo?:
Clonar el repositorio
Abrir el proyecto en NetBeans
Ejecutar el archivo principal
